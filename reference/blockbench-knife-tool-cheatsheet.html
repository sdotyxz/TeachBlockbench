Knife Tool

Use Knife when one existing surface should become smaller editable pieces. For a beginner cube workflow, think: split the cube first, then color or move the pieces.

Source

Blockbench's source defines Knife Tool as an Edit-mode main tool for cutting mesh faces into smaller pieces, and its cube path splits a selected cube into two cube elements. [Knife Tool source] [English label]

Two behaviors

Selected thingWhat Knife doesBeginner use
Cube Splits the cube into two cube elements along a straight plane. Make planks, tiles, counter panels, cut marks, layered parts.
Mesh Adds points and lines, then rebuilds mesh faces into smaller faces when applied. Save for later: irregular shapes, advanced face editing, mesh topology.

Cube split recipe

  1. Select one cube in Edit Mode

    Knife is an Edit Mode tool. If many cubes are selected, pick the one you want to split first so the result is predictable.

  2. Choose Knife Tool

    Use the surgical-knife icon in the main toolbar. Your cursor becomes a crosshair.

  3. Click the face where the split should pass

    The first click sets the split position. Use a large face, usually the top face, so you can see the preview clearly.

  4. Move until the preview plane points the right way

    The second point chooses the split direction. Watch the thin preview sheet, not the cursor.

  5. Click again to apply

    The single cube becomes two cubes. Rename them right away if you will keep editing both.

Modifier keys

KeyUseToday
Ctrl / CmdSnap to pixels.Use when the cut should land cleanly on the pixel grid.
ShiftSnap to center.Use for a middle cut; avoid it for uneven handmade planks.
EscCancel active Knife operation.Use if the preview plane points the wrong way.
EnterApply active mesh Knife operation.Mostly for mesh cuts; cube cuts apply on the second click.

Troubleshooting

SymptomLikely causeFix
Knife icon is missingYou are not in Edit Mode, or the toolbar is customized.Switch to Edit Mode or use Tools / Action Control and search Knife Tool.
The cut goes the wrong directionThe second click chose the other axis.Undo, repeat, and watch the preview plane before the second click.
Nothing obvious changedThe two new cubes are still touching and have the same color.Open the Outliner, select each piece, rename it, and color one side slightly darker.
A warning says different faces cannot be cutYou are using mesh Knife across unsupported faces.For now, use cube splitting only; mesh Knife gets its own lesson later.
