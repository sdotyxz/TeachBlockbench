Reference - pixel art for tiny 3D textures

Pixel Art Texture Cheatsheet

A small texture does not need many details. It needs readable groups of pixels: base color, shadow, highlight, and a few material marks.

The five rules

  1. Use a tiny canvas on purpose. Start with 16 x 16 or 32 x 32 for small Blockbench props. Increase only when the prop truly needs more readable faces.
  2. Use three values per material. Base, shadow, highlight. If you need more, make sure the reason is visible from a distance.
  3. Paint clusters, not dust. A cluster is a small connected group of same-color pixels. Many isolated one-pixel specks usually read as noise.
  4. Check the thumbnail. Zoom out until the prop is small. If the material reads, keep it. If it buzzes, delete details.
  5. Let the model do shape work. Do not draw every edge. Use texture pixels to clarify material, direction, and small accents.
