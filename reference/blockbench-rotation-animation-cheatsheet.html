Rotation Animation Cheatsheet

A beginner-safe map for animating a small part that pivots: one group, a centered pivot, Rotation keyframes, and a matching first/last pose for looping.

Mental model

Position keyframes move a part through space. Rotation keyframes turn a part around its pivot. If the pivot is wrong, the rotation will swing around the wrong point.

Axis rule

If your timer face is built like a flat front-facing square, the needle usually spins on Z. If it turns like a door, you are probably rotating on the wrong axis or the model is facing a different direction.

Troubleshooting

SymptomLikely causeFix
The whole timer rotates.The timer body is selected or grouped with the needle.Select only the needle group before adding Rotation keyframes.
The needle orbits around the dial.The pivot is not at the dial center.Move the needle group's pivot to the center, then redo the keyframes.
The needle flips like a door.The wrong rotation axis is keyed.Undo and try the Z axis for a front-facing timer.
The loop pops at the end.The final rotation differs from the first.Set 1.00s to the same value as 0.00s.
