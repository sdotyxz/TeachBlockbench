Seam Tool

Use Seam Tool when a mesh needs a cleaner generated texture layout. Think: choose where the texture is allowed to cut or stay joined.

Source

Blockbench's source defines Seam Tool as an Edit-mode tool for UV seams on mesh edges, with three seam modes: Auto, Join, and Divide. The texture generator treats Divide as a hard stop between faces, while Join overrides the usual angle-based split. [Seam Tool source] [Template generator source]

The three modes

ModeMeaningBeginner use
Auto Let Blockbench decide whether adjacent mesh faces should stay in the same UV island. Default. Use when you do not care where the texture layout splits.
Divide Force a cut between adjacent faces when generating a texture template. Put the cut on a hidden back or bottom edge.
Join Force adjacent faces to remain connected when generating a texture template. Use only when the generated strip breaks into too many islands.

Beginner recipe

  1. Use a mesh, not a cube

    Seam Tool is available for mesh elements. Add a mesh from the Outliner's add menu, or convert a cube to mesh when you specifically need mesh UV control.

  2. Choose Seam Tool in Edit Mode

    The tool switches the mesh into edge selection and solid view so the edges are easier to read.

  3. Select the edge where a cut should hide

    For bottles, cups, jars, and rounded props, choose a vertical back edge. For boxes, choose a bottom or rear edge.

  4. Set Select UV Seam to Divide

    The edge should tint red or orange. That is the "cut here" instruction for the next texture template generation.

  5. Generate or append a texture template

    Seams influence the template layout. They do not repaint an existing texture by themselves.

Color cues

Edge colorMeaningAction
Red / orangeDivide seamGood for hidden cuts.
GreenJoin seamGood for keeping side faces as one strip.
Neutral grid colorAuto seamBlockbench decides.
White-ish selected edgeSelected edgeSet the seam mode from the toolbar.

Troubleshooting

SymptomLikely causeFix
Seam Tool is missingNo mesh exists, or you are not in Edit Mode.Add Mesh or Convert to Mesh, then return to Edit Mode.
The seam changed color but the texture did not changeSeams affect generated templates, not already-painted pixels.Create a new Texture Template or append the mesh to a template.
The label appears on the backThe painted area is on the wrong part of the UV island.Move the label pixels on the texture or move the UV island in the UV panel.
The side band is split into many piecesAuto decided to divide too many angled faces.Set important vertical edges to Join, leaving one hidden Divide seam.
