Selection Mode

Selection Mode changes what a click in the viewport grabs. For meshes, it lets you work at object, connected-cluster, face, edge, or vertex level.

Source

Blockbench's source lists Selection Mode options as Object, Cluster, Face, Edge, and Vertex. Its click handling for Cluster starts from the clicked face and recursively selects faces that share vertices with it. [Selection Mode source] [Cluster click source]

The five modes

ModeWhat a click selectsUse it for
ObjectThe whole cube, mesh, or element.Normal beginner editing and Outliner work.
ClusterAll connected faces inside one mesh chunk.Moving, resizing, rotating, or splitting one disconnected part of a merged mesh.
FaceOne surface at a time.UV placement, labels, and local texture fixes.
EdgeOne mesh edge at a time.Seam Tool, Loop Cut, and edge-level fixes.
VertexIndividual corner points.Advanced mesh shaping; use carefully.

Cluster rule

Plain meaning

Cluster = connected faces. If faces share vertices, Cluster treats them as one selectable island. If two shapes live in the same mesh but do not share vertices, Cluster can select one shape without selecting the other.

Beginner recipe

  1. Use meshes

    Cluster is useful when you are editing mesh faces. If everything is still simple cubes, Object mode is usually enough.

  2. Create separate mesh pieces

    Add two or more mesh primitives, such as spheres or cylinders, and position them near each other.

  3. Merge them only when needed

    Use Mesh → Merge Meshes to keep them in one mesh element while preserving disconnected face islands.

  4. Switch to Cluster

    With the merged mesh selected in Edit Mode, change Selection Mode to Cluster.

  5. Click one face

    The connected chunk should highlight. Move, resize, rotate, or split it as one part.

Troubleshooting

SymptomLikely causeFix
Cluster selects the whole objectAll faces are connected, like one sphere or cylinder.This is correct. Use Cluster on a mesh with disconnected chunks.
Cluster grabs two chunksThe chunks share at least one vertex.Undo any accidental vertex merge, or separate the geometry before merging meshes.
Selection Mode is missingNo mesh is selected, or you are not in Edit Mode.Select a mesh element, switch to Edit Mode, then look at the toolbar.
You can no longer select the whole meshYou stayed in Cluster, Face, Edge, or Vertex mode.Switch back to Object before normal whole-element edits.
