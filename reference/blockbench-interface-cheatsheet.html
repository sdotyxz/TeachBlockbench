Blockbench Interface & Navigation

The 20% of the interface you'll use 80% of the time. Designed to print on one page.

How to use this

Don't memorize it — keep it open in a second tab. The moves become automatic after a few sessions. Everything here is from the official Blockbench Wiki — Overview & Tips.

Get Blockbench

OptionWhereWhen to use
Web app (no install)web.blockbench.netRight now — fastest way to start. Great for learning.
Desktop appblockbench.net/downloadsLater — Win/Mac/Linux. Best for plugins & exporting a finished pack.

Free and open-source, for any project, forever.

Start a model

  1. New project

    Top menu → File → New.

  2. Pick Generic Model

    This is the format for normal game assets (the Minecraft formats add rules you don't want). Always choose Generic Model for this course.

  3. Name & create

    Give it a name (e.g. crate) and confirm. You now have a blank 3D workspace.

Move the camera (viewport navigation)

You're not moving the model — you're moving your view of it. Three moves do everything:

MoveDo thisThink of it as
ZoomScroll wheel (works on any setup)Step closer / further away
Rotate / orbitLeft-click + drag on empty spaceWalk around the object
PanRight-click + dragSlide the whole view sideways
If you use a mouse with a middle button

Alternative defaults: middle-click = rotate, Shift+middle = pan, Ctrl+middle = zoom. Scrolling to zoom always works. You can also load Blender/Maya keymaps in Preferences later.

Lost your model? Re-center the view

Select a cube, then run Center View on Selection — it snaps the camera back onto whatever's selected. It's in the viewport right-click menu. Heads-up: the default keymap assigns it no shortcut — set your own in Preferences → Keybindings (search "Center View on Selection"); F is the natural pick (it's what the Maya keymap uses). Hold Shift while triggering it to rotate only (no zoom), or Ctrl to zoom in close. [Wiki]

WASD fly navigation (optional plugin · desktop app)

Move through the scene like a first-person game instead of orbiting. Install the WASD Controls plugin (File → Plugins), then enable View menu → WASD Movement. The viewport must be focused (not a text field). Best for big scenes, not single props — your mouse moves above are faster for posing one object.

ActionDefault key
Move forward / backW / S
Move left / rightA / D
Move up / downSpace / Shift
Faster / slower (hold)Ctrl / Alt

All eight are remappable in Preferences → Keybindings. Tip: turn on "WASD Hold Right Mouse Button" in settings so W/A/S/D only fly while right-mouse is held — that keeps the letters free for naming cubes. [Plugin source]

The 5 core tools

Pick a tool from the toolbar, then drag the colored arrows/handles on the selected cube.

ToolWhat it does
MoveSlides an element along X / Y / Z.
ResizeStretches or shrinks an element's size.
RotateTurns an element around a point.
PivotMoves the point something rotates around.
Vertex SnapGlues one cube's corner exactly onto another's.
The one shortcut to remember

Press Space (or double-click in the viewport) to flip between Move ⇄ Resize (and Rotate ⇄ Pivot). For exact numbers, type values straight into the Element panel instead of dragging.

The panels around the screen

PanelWhat lives there
OutlinerThe list of every cube & group in your model. Rename and organize here.
ElementExact numbers for the selected cube: Position, Size, Pivot, Rotation.
TexturesYour texture images. Create or import them here.
UVHow a texture wraps onto the cube's faces (covered in a later week).
ColorColor picker & palette, used in Paint mode.

First-words glossary

TermPlain meaning
Low-polyA model made of few, chunky flat surfaces — the blocky Overcooked look. The opposite of smooth/realistic.
Cube / ElementA single box. Models are built by combining boxes. "Element" = any one piece.
Generic ModelBlockbench's neutral project type for ordinary 3D assets. Your default.
TextureThe flat image that gives a model its color/detail, painted onto its surfaces.
UV mapThe "unfolded" layout that says which part of the texture lands on which face.
PivotThe anchor point a piece rotates around.
.glb / .gltfCommon game-ready export formats — what a buyer drops into a game engine.
