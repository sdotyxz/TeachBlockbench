The Course Palette: Endesga64

The course now uses Endesga64: a 64-color pixel-art palette by ENDESGA. Every prop should pull from these colors only. The small starter set below keeps beginner exercises focused, while the full palette gives you room for richer materials later.

How to use this

When a part is wood, use the same wood. When a part is metal, use the same metal. Type the HEX code into Blockbench's Color Bar, or click the saved swatch in the Palette tab. Blockbench also includes a built-in Endesga 64 palette under its palette loading options, so load that when available and use these role names as course shorthand.

Why a limited palette sells

  • Cohesion — repeated colors signal "these belong together," which is exactly what a buyer wants from a pack.
  • Speed — you never stop to invent a color; you reach for a saved swatch.
  • Mood — Endesga64 gives the course bright food colors, usable neutral ramps, and strong pixel-art contrast without breaking pack cohesion.
Rule of thumb

Add a color from the full Endesga64 set only when you can name the material job it does: shadow, base, highlight, accent, food color, metal, or readable UI mark. Constraint is the style.
