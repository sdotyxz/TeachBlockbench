The Course Palette

Six warm, saturated colors. Every prop in the pack uses only these. A small, shared palette is the cheapest way to make separate objects look like one designed set.

How to use this

When a part is wood, use the same wood. When a part is metal, use the same metal. Type the HEX code into Blockbench's Color Bar, or click the saved swatch in the Palette tab. Never eyeball a "close enough" brown — matching exactly is what reads as a set.

The six colors

Cream plates, ceramic, dishware #F3E6C4
Wood crate, board, handles #C68B45
Dark Wood wood shadow + edges #7A4E24
Steel pot, knife, spoon, metal #AEB8BE
Tomato warm accent / pop #D94A36
Leaf garnish, fresh accent #5FA63E

Why a limited palette sells

  • Cohesion — repeated colors signal "these belong together," which is exactly what a buyer wants from a pack.
  • Speed — you never stop to invent a color; you reach for one of six.
  • Mood — warm + saturated + few colors reads cheerful and cartoony, the Overcooked feel.
Rule of thumb

Add a seventh color only when you can name a real reason. Constraint is the style — a tight palette is a feature of the pack, not a limitation.
