Runtime Flows

  1. Choose a format, usually Generic Model.
  2. Add cubes, meshes, groups, locators, null objects, or splines.
  3. Transform with Move, Resize, Rotate, Pivot, Stretch, Inflate, Mirror Modeling, and Vertex Snap.
  4. For mesh modeling, expand into Knife, Loop Cut, Extrude, Inset, Solidify, Merge/Split, Seam, and weight tools.
  1. Select faces or UV vertices.
  2. Use UV grid, rotation, maximize, auto, relative auto, project from view, mirror, reset, apply all.
  3. For mesh UVs, use merge UV vertices, connect faces, cycle, snap to pixels, and move texture with UV.
  1. Create/import textures and palettes.
  2. Paint with Brush, Copy Brush, Bucket, Eraser, Color Picker, Draw Shape, Gradient, Selection, and Move Layer.
  3. Control brush shape, blend mode, fill mode, opacity, softness, mirror painting, alpha lock, grid, tiled view, onion skin, and layers.
  1. Enter Animate or Pose mode.
  2. Add an animation, set length and snapping, add animators and keyframes.
  3. Use interpolation, loop playback, mirror animating, graph editor, onion skin, controllers, sounds/particles only when the project needs them.
  1. Use Display and Preview tools for readable renders.
  2. Save BBModel, save textures, export GLTF/OBJ/FBX/STL or platform-specific formats.
  3. Package model files, textures, screenshots, license, and itch.io page assets.
