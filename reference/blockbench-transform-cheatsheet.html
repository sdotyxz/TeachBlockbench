Reference — keep this open while rotating parts

Move, Resize, Rotate, Pivot

The small set of transformations that turn boxes into props. Use exact numbers first; drag handles once you understand what should happen.

Source

Blockbench's official overview identifies X as width, Y as height, and Z as length; the Element panel exposes Position, Size, Pivot Point, and Rotation sliders; and Edit mode includes Move, Resize, Rotate, Pivot Tool, and Vertex Snap. [Blockbench Wiki]

The four transformations

ToolChangesUse it when
MovePositionA part is the right shape but in the wrong place.
ResizeSizeA part is too wide, tall, short, thick, or thin.
RotateRotationA part should sit at an angle instead of following the grid.
PivotRotation pointA part rotates from the wrong hinge or joint. Save this for later.
Beginner rule

When a lesson gives numbers, type them into the Element panel. Typed values are calmer than dragging because you can see exactly what changed.

Axes in plain words

AxisPlain meaningToday's use
XWidth, left to rightMake a plate or spoon wider.
YHeight, up and downTurn a utensil diagonally on a tabletop.
ZLength, front to backMake a tray deeper or a handle longer.

Rotation recipes

GoalTry thisWatch for
Diagonal spoon on plateRotation Y -35The spoon turns across the tabletop.
Opposite diagonalRotation Y 35Same angle, mirrored direction.
Back to straightRotation 0 0 0The part returns to the grid.
Matching two partsUse the same number on bothSeparate cubes read as one object.

Troubleshooting

SymptomLikely causeFix
The model angle changed, but the part did notYou orbited the camera.Select the cube and change Element → Rotation.
The spoon flickers on the plateTwo faces occupy the same surface.Move the spoon slightly upward.
The bowl and handle do not line upTheir Rotation values differ.Type the same Y Rotation value on both parts.
A part swings from a strange pointThe pivot is not where you expect.Ignore for now; use Move after Rotate. Pivot gets its own lesson later.
