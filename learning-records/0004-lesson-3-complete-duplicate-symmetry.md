# Lesson 3 complete: duplicate for symmetry

Completed 2026-06-25 (self-reported by the learner). The learner built and rendered a cooking pot
with a body, overhanging rim, and two matching handles, then completed the third daily lesson.

## What is now established

- Duplicate can copy a finished part instead of rebuilding it by hand.
- A duplicated part can be moved across a model to create matched symmetry.
- Clear Outliner names (`body`, `rim`, `handle-l`, and `handle-r`) keep a multi-part prop readable.
- The kitchen set now contains three props: crate, chopping board with knife, and cooking pot.
