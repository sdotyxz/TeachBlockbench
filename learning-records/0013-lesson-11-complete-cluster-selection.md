# Lesson 11 complete: Cluster Selection tomato trio

Completed 2026-07-03 (completion requested by the learner). The learner built a
low-poly tomato trio and practiced Selection Mode: Cluster as the main mesh
editing idea: selecting one connected tomato chunk inside a multi-piece produce
prop without selecting every face by hand.

## What is now established

- Cluster Selection is a selection mode for connected mesh face islands.
- A merged or multi-island mesh can still contain separately selectable connected chunks.
- Cluster is useful when one whole connected mesh piece needs a small move, resize, or rotation.
