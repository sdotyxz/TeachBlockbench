Publishing Setup

  1. Push this repo to GitHub

    Commit the course files and push the branch you want GitHub Pages to publish.

  2. Enable GitHub Pages

    In GitHub, open Settings → Pages, then choose Deploy from a branch.

  3. Pick the source

    Select your main branch and the root folder /. GitHub will publish this index.html as the site homepage.

  4. Open it from your phone

    After GitHub finishes publishing, open the Pages URL on your phone. Use the Continue box above to jump straight into the current lesson.
Public site note

GitHub Pages is internet-accessible. Keep private notes, unpublished screenshots, or anything sensitive out of the published branch.
